# フロントエンド構成

## HTML
- `index.html`：タイマー表示、進捗サマリー、作業/休憩時間カスタマイズUI、ボタン群

## CSS
- `style.css`：レスポンシブ対応、タイマー・進捗・カスタマイズUIのスタイリング

## JavaScript
- `timer.js`：
    - PomodoroTimerクラスでタイマー状態・進行管理
    - localStorageによる状態・設定保存/復元
    - 進捗円グラフ描画
    - UI要素の更新・イベントハンドラ
- `beep.js`：
    - ビープ音再生（2音）
    - AudioContextの初期化・ユーザー操作時の有効化

## 主要UI要素
- タイマー表示（分:秒）
- 進捗円グラフ
- 開始・一時停止・リセットボタン
- 作業/休憩時間カスタマイズ
- 今日の完了回数・集中時間サマリー

## 状態管理
- localStorageでタイマー状態・設定を保存
- ページ再読込時も状態復元
