# フロントエンド構成

## JavaScript モジュール

### timer.js
- ポモドーロタイマーのロジック・状態管理・UI更新を担当。
- `PomodoroTimer` クラスで作業/休憩の切替、セット管理、localStorage保存、進捗円描画などを実装。
- 設定値（作業/休憩時間）はカスタマイズ可能で、保存ボタンでlocalStorageに保存。
- タイマーの開始/一時停止/リセットボタン、進捗円、今日の進捗表示（完了数・集中時間）を制御。

### beep.js
- ビープ音再生（2音: 高→低）を提供。
- AudioContextの自動再生制限対策も実装。

### timer.test.js
- timer.jsのロジックテスト（Jest形式）。

## HTML テンプレート

### index.html
- メイン画面。タイマー表示、進捗円、カスタマイズUI、進捗サマリーを含む。
- 各種ボタン・入力欄・進捗表示要素にID付与。

## CSS

### style.css
- レスポンシブ対応。
- タイマー・進捗円・カスタマイズUI・進捗サマリーのスタイル定義。
