# APIリファレンス

このアプリケーションはFlaskベースで、以下のエンドポイントを提供します。

## ルートエンドポイント

### `GET /`
- 概要: ポモドーロタイマーのメイン画面を返します。
- レスポンス: HTML (テンプレート `index.html`)
- テンプレート変数:
    - `work_minutes` (int): 作業時間（分）。デフォルト1。
    - `initial_seconds` (int): 初期秒数。デフォルト0。
- ステータスコード: 200
