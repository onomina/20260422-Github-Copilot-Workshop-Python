# アーキテクチャ概要

本アプリはシンプルな構成で、以下のレイヤーで構成されています。

- **バックエンド（Flask）**
    - `/` ルートのみ実装。index.htmlを返却。
    - 今後の拡張でAPI追加可能。
- **フロントエンド（HTML/CSS/JS）**
    - `index.html`：UIの骨組み。
    - `style.css`：レスポンシブ対応含むデザイン。
    - `timer.js`：タイマー制御・状態管理・進捗描画・設定保存。
    - `beep.js`：ビープ音再生。

状態管理は基本的にlocalStorageで行い、サーバー側のデータ保存は未実装です。
